## 《最南幻想》操作手册

### 1. 系统要求

| 项目           | 最低配置                                      | 推荐配置                                         |
| :------------- | :-------------------------------------------- | :----------------------------------------------- |
| 操作系统       | Windows 10 (64-bit) / macOS 11 / Ubuntu 20.04 | Windows 11 / macOS 13 / Ubuntu 22.04             |
| 处理器         | Intel i5-7200U / AMD Ryzen 3 1200             | Intel i5-10400 / AMD Ryzen 5 3600                |
| 内存           | 4 GB RAM                                      | 8 GB RAM                                         |
| 显卡           | 支持 OpenGL 3.3 的集成显卡                    | 支持 Vulkan 1.2 的独立显卡                       |
| 存储空间       | 500 MB 可用空间                               | 1 GB 可用空间                                    |
| AI 服务 (可选) | 无 (可使用云端 API)                           | 已安装 Ollama，并下载 `qwen2.5:7b-instruct` 模型 |

### 2. 安装与启动

1. 从大赛官网或 GitHub Releases 下载对应平台的压缩包。
2. 将压缩包解压到任意目录。
3. **如果使用本地 AI（Ollama）**：
   - 请确保已安装并启动 Ollama（终端执行 `ollama serve`）。
   - 确保已下载模型：`ollama pull qwen2.5:7b-instruct`。
4. **如果使用云端 API**：
   - 在 游戏设置 > AI设置 中填入您使用的Base URL、Model 和 API 密钥。
   - 请遵守 API 密钥保密协议。
5. 双击 `最南幻想.exe` (Windows) 或 `最南幻想.app` (macOS) 启动游戏。

### 3. 基本操作

| 操作           | 说明                                                         |
| :------------- | :----------------------------------------------------------- |
| **推进对话**   | 鼠标左键点击 / 按空格键 / 回车键                             |
| **选择选项**   | 鼠标左键点击选项按钮                                         |
| **打开好感度** | 鼠标右键 / 按 `A` 键                                         |
| **快速读档**   | 点击 HUD 上的“快速读档”按钮                                  |
| **自动模式**   | 点击“自动”按钮，文本显示完后自动前进                         |
| **快进模式**   | 点击“快进”按钮，长对话1秒跳过，普通对话3倍速自动前进         |
| **打开存档**   | 点击“存档”按钮，选择槽位                                     |
| **打开读档**   | 点击“读档”按钮，选择存档                                     |
| **返回主界面** | 点击“主界面”按钮，可选择保存进度                             |
| **设置**       | 点击“设置”按钮，可调节音量、文本速度、自动速度，进行AI有关设置等 |

### 4. 存档位置

- **Windows**: `%APPDATA%\Godot\app_userdata\nju_dream\saves\`
- **macOS**: `~/Library/Application Support/Godot/app_userdata/nju_dream/saves/`
- **Linux**: `~/.local/share/godot/app_userdata/nju_dream/saves/`

### 5. 常见问题

- **Q: 游戏打开后黑屏？**
  A: 请检查显卡驱动是否已更新，并确认操作系统满足最低要求。
- **Q: AI 无响应？**
  A: 若使用本地 Ollama，请确保服务已启动 (`ollama serve`)，模型已下载。若使用云端 API，请检查  游戏设置 > AI设置 中的 API Key 是否正确，以及网络是否通畅。
- **Q: 存档无法读取？**
  A: 请检查游戏版本是否与存档版本一致。不同版本间的存档可能不兼容。
- **Q: 按钮音效不响？**
  A: 请在设置界面中检查“按钮音效”开关是否打开，并确认 SFX 音量不为 0。
